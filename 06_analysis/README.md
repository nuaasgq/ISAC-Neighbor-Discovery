# Analysis Workflow

This directory stores scripts and lightweight outputs for turning simulation runner
results into paper-facing tables and figures.

## Input Contract

The analysis script expects one or more runner output directories, or a sweep root
that contains runner output subdirectories. Each run directory should contain:

- `per_episode_summary.csv`
- `per_slot_metrics.csv` optional, but recommended

These files are generated by `05_simulation/run_smoke.py` or
`isac_nd_sim.runner`.

## Command

Analyze a single run directory:

```powershell
python 06_analysis/scripts/analyze_dynamic_results.py `
  05_simulation/results_raw/mobile_smoke `
  --output 06_analysis
```

Analyze a sweep root containing many run subdirectories:

```powershell
python 06_analysis/scripts/analyze_dynamic_results.py `
  05_simulation/results_raw `
  --output 06_analysis
```

If `matplotlib` is not installed, the script still writes tables and records
figure skips in `figures/figure_manifest.json`. To force table-only analysis:

```powershell
python 06_analysis/scripts/analyze_dynamic_results.py `
  05_simulation/results_raw `
  --output 06_analysis `
  --no-plots
```

## Outputs

Tables:

- `tables/protocol_summary.csv`: protocol-level means, standard deviations,
  medians, extrema, and 95% confidence half-widths.
- `tables/protocol_mobility_summary.csv`: protocol-level summary split by
  mobility model.
- `tables/slot_protocol_summary.csv`: per-slot protocol trajectories for LCC,
  `lambda2`, discovery progress, empty-scan ratio, and collisions.
- `tables/source_runs_manifest.csv`: input run directories and row counts.

Figures:

- `figures/protocol_empty_scan_ratio.png`
- `figures/protocol_discovery_rate.png`
- `figures/protocol_p95_discovery_delay.png`
- `figures/protocol_p99_discovery_delay.png`
- `figures/protocol_lcc_ratio.png`
- `figures/protocol_lambda2.png`
- `figures/figure_manifest.json`

The default figure set is intentionally minimal: it supports early manuscript
drafting and sanity checks without locking the final paper style.
